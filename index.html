Umbrella

a Programming Language
What is it:

Umbrella is a programming language influenced by C and Rust, which follows 3 main pillars:

  1. Simple Compiler Source - 70% of the performance in 10% the code (inspired by QBE)

  2. Fast Prototyping - less friction when programming & discourage premature optimization

  3. Compiler Tooling - make it easy to experiment with program analysis tools
